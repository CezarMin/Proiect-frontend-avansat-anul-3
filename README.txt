Aplicatia are
- mai multe rute: /employees, /employees/add, /employee/edit?guid=.
- backend-ul pentru add, getemployees este functional, pentru edit nu.
- atat add-employee si edit-employee se folosesc de employee-form, edit-employee transmitand un employee cu datele angjatului, add-employee transmitand un angjat cu date goale (folosind @Input()).
- componentele comunica intre ele folosind @Input().
- form-uri de add si edit.

Backend: .net core cu sql server.
